# メッセージカード画像生成アプリ 仕様書

---
# 1. 概要
ユーザー入力からメッセージカード画像を生成し、PNG保存できるフロントエンドWebアプリ。

---
# 2. 技術スタック
React / TypeScript / Vite / HTML Canvas
バックエンド禁止

---
# 3. 入力項目
タイトル（30文字, デフォルト Thanks Card）
宛先（To: xxx）
メッセージ（200文字, 折り返し, 最大5行）
日付（mm/dd, 起動時に今日の日付を自動セット）
差出人（Fm: xxx）
左下画像（PNG/JPG/WebP, 5MB以下）
背景色（カラーピッカー, デフォルト #FFFACD）
タイトルフォント（日本語Google Fonts / 欧文Systemフォントから選択）
本文フォント（日本語Google Fonts / 欧文Systemフォントから選択）
フォントサイズ（Title / To / Message / Date / From を各8〜60pxで個別指定）

---
# 4. 出力
PNG
450x340
message-card_YYYY-MM-DD.png

---
# 5. カード配置
上中央: タイトル
左上: To
中央: メッセージ（折り返し, 最大5行）
右下: 日付 + Fm
左下: 画像 contain

---
# 6. 画面UI仕様
デスクトップ: 左フォーム / 右プレビュー（2カラム）
モバイル: 縦積み
入力即時プレビュー更新
ダークUI + 明るいカード
角丸 8px

---
# 7. 受け入れ条件
即時プレビュー
PNG保存可
レスポンシブ対応
Vercel公開可
